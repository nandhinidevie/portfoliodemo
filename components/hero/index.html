Overview

-- ds.hero contains, a image to the right side and information to the left side of it. If you want to use it then use below given example code and add it inside a .ftd file.

Invocation:

-- ds.hero: John Doe
tag: Hello there...
designation: I Am Passionate Developer!
avatar: $assets.files.assets.me.jpg
cta-primary: My Work
cta-primary-link: index.html
cta-secondary: Hire Me
cta-secondary-link: index.html

We are an award-winning strategic design company that provides 
consultancy services and help you create outstanding digital products.
