Overview:

-- ds.skills contains, a image to the right side and information to the left side of it. If you want to use it then use below given example code and add it inside a .ftd file.

Invocation:

-- ds.skills: MY SKILLS
tag: A LEAD UX & UI DESIGNER BASED IN CANADA
languages: $common.languages
achievements: $common.achievements
Lang:
ftd

Properties:

Below are details of all properties of -- ds.skills component.
title
type:
Caption (required)
This is a default property of the skills component. It's used to add a title to skills component.
tag
type:
string (required)
This is a default property of the skills component. It's used to add a tag to skills component.
languages
type:
list (required)
This is a default property of the skills component. It's used to add a list of languages that the user wants to add to skills component.
achievements
type:
list (required)
This is a default property of the skills component. It's used to add a list of achievements that the user wants to add in achievements section.
