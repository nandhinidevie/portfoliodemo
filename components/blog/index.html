tag
type:
Caption (required)
This is a default property of the blog component. It's used to add a tag to the blog.
publishing-date
type:
Caption (required)
This is a default property of the blog component. It's used to add a publishing-date of the blog.
src
type:
ftd.image-src (required)
This is a default property of the blog component. Using src attribute cover image can be added.
